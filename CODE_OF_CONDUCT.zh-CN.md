# Contributor Covenant 行为准则

> 语言： [English](CODE_OF_CONDUCT.md) | **简体中文**

## 我们的承诺

我们承诺让每个人都能免于骚扰地参与 OMP API Manager 社区，不因年龄、体型、残障、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向而受到区别对待。

## 我们的标准

有助于营造积极环境的行为包括：

- 对不同观点和经历展现同理心与尊重。
- 以友善方式给予和接受建设性反馈。
- 关注最有利于社区和用户的结果。
- 尊重隐私，并谨慎处理安全敏感信息。

不可接受的行为包括骚扰、歧视性语言、人身攻击、未经允许发布私密信息，以及任何在专业社区中可被合理视为不当的行为。

## 执行

项目维护者负责阐明和执行这些标准。请通过本仓库的私密 GitHub 安全咨询报告不当行为。报告会被及时审查，并尽可能保密处理。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1。
