## 报告漏洞

不要在公开 Issue 中报告疑似凭据泄露、配置覆盖、网关认证绕过或本地数据泄露。请通过本仓库的私密 GitHub 安全咨询提交复现步骤，并排除真实凭据。

我们将在七天内确认收到报告。报告中不得包含 API Key、授权头、提示词或用户配置内容。

请尽可能提供受影响版本、影响范围、最小复现步骤和安全的概念验证。我们会与报告者协调修复与披露时间表。
