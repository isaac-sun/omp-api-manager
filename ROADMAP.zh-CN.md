# 路线图

> 语言： [English](ROADMAP.md) | **简体中文**

## v0.1.0 — 已发布

- [x] OMP 16.x 检测以及读写兼容性边界
- [x] YAML 事务安全机制与备份
- [x] Keychain 服务抽象
- [x] 服务商 CRUD 与 Anthropic 兼容适配器
- [x] 完整连接测试与安全 OMP 应用流程
- [x] 本地流式网关
- [x] SQLite 用量存储、仪表盘和 CSV/JSON 导出
- [x] UI 与网关集成测试

## v0.2.0 — 已发布

- [x] 安全导入 `newapi_channel_conn`
- [x] 详细的多模型服务商配置
- [x] 可选择的 OMP API 模式
- [x] 不复制凭据的服务商复制功能
- [x] 中英双语文档与可下载的 DMG Release

## v0.3.0 — 下一版本

- [x] 首要功能：由用户主动检查 GitHub 最新正式 Release
- [x] 标准 macOS **检查更新…** 菜单和“软件更新”设置页
- [x] 严格的语义版本比较、响应验证、ETag 复用和限流处理
- [x] 仅跳转到官方 GitHub Release 页面，不自动下载或安装
- [ ] Developer ID 签名、公证和带独立签名的自动更新
