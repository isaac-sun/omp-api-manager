## Reporting a vulnerability

Do not report suspected credential exposure, configuration overwrite, gateway authentication bypass, or local data disclosure in a public issue. Open a private GitHub security advisory for this repository with reproduction steps that exclude real credentials.

We aim to acknowledge reports within seven days. Never include API keys, authorization headers, prompts, or user configuration contents in a report.

Please include the affected version, impact, minimal reproduction, and a safe proof of concept where possible. We will coordinate a fix and disclosure timeline with the reporter.
