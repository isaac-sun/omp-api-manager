# OMP API Manager v0.1.0

Released 2026-07-11.

## First developer release

This is the first public source release of OMP API Manager for macOS 14+ and Swift 6. It provides a local-first workflow for secure custom-provider management, a localhost-only gateway, and private usage inspection.

## Included

- OMP 16.x installation discovery and safe configuration integration.
- Atomic YAML updates with backups and conflict detection.
- macOS Keychain-backed provider credentials.
- OpenAI-compatible and Anthropic-compatible provider setup, model fetch, and connection testing.
- Localhost-only gateway with a separate local token and SSE forwarding.
- Sanitized SQLite usage tracking plus CSV and JSON export.
- A native SwiftUI interface and 22 automated tests.

## Installation

Download `OMP-API-Manager-0.1.0-macos-arm64.dmg` from the release assets, open it, and drag the app to Applications. The build is ad-hoc signed but not Developer ID notarized. macOS may show an unidentified-developer warning; only open it if you downloaded it from this repository's official release page.

You can also clone the tagged source, open `Package.swift` in Xcode, and run the `OMPAPIManager` scheme; alternatively run `swift run OMPAPIManager` from Terminal.

## Security notes

API keys remain in the macOS Keychain. The app does not persist prompts, responses, authorization headers, or API keys. See [Security](../../SECURITY.md) and [Privacy](../../PRIVACY.md).

## Reproducible packaging

On an Apple Silicon Mac with Xcode installed, run `scripts/package-dmg.sh 0.1.0`. The command creates a DMG in `dist/` and prints its SHA-256 checksum.
