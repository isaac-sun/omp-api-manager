# OMP API Manager v0.3.0

> Language: **English** | [简体中文](../zh-CN/releases/v0.3.0.md)

Released 2026-07-11.

## Manual update checks

This release adds a native, privacy-conscious **Check for Updates…** workflow for macOS. It is available from the application menu and the Software Update settings page.

- Checks only when the user requests it; there is no launch-time background polling.
- Requests the latest stable release from the fixed public GitHub endpoint with no GitHub token, cookies, device identifier, OMP configuration, provider metadata, API keys, or usage data.
- Compares versions using strict semantic-version rules, including correct handling for development and prerelease builds.
- Reuses ETags during the app session, handles rate limits clearly, and rejects unexpected response origins, malformed releases, prereleases, and oversized responses.
- Constructs the official release URL locally from a validated tag rather than trusting a URL returned by the network.

## Installation behavior

When an update is available, the app opens this repository's official GitHub Release page. It does not download, mount, replace, or execute an update automatically.

That boundary is intentional: current DMGs are ad-hoc signed and not Developer ID notarized. Automatic updates will require Developer ID signing, notarization, and an independent update signature.

## Installation

Download `OMP-API-Manager-0.3.0-macos-arm64.dmg` and its `.sha256` file from the release assets. Put both files in the same directory and run:

```sh
shasum -a 256 -c OMP-API-Manager-0.3.0-macos-arm64.dmg.sha256
```

Open the verified DMG and drag the app to Applications. The build supports Apple Silicon and requires macOS 14 Sonoma or later.

## Validation

The release source passes a strict Swift build and all 31 automated tests. The packaging process verifies the application signature and starts the finished app before creating the DMG.

## Reproducible packaging

On an Apple Silicon Mac with Xcode installed, run `scripts/package-dmg.sh 0.3.0`. The version argument must match `Sources/OMPAPIManagerApp/Resources/AppVersion.txt`.
