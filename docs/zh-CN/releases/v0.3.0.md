# OMP API Manager v0.3.0

> 语言： [English](../../releases/v0.3.0.md) | **简体中文**

发布于 2026-07-11。

## 手动检查更新

此版本新增原生、重视隐私的 macOS **检查更新…** 流程，可从应用菜单和“软件更新”设置页使用。

- 只在用户主动请求时检查，不会在应用启动时后台轮询。
- 只访问固定的公开 GitHub 最新正式 Release 端点；不发送 GitHub Token、Cookie、设备标识、OMP 配置、服务商元数据、API Key 或用量数据。
- 使用严格的语义版本规则比较版本，并正确处理开发版本和预发布版本。
- 在应用运行期间复用 ETag，明确处理限流，并拒绝异常响应来源、格式错误的 Release、预发布版本和过大的响应。
- 根据经过验证的标签在本地构造官方 Release URL，而不是直接信任网络返回的 URL。

## 安装行为

发现新版本后，应用会打开本仓库的官方 GitHub Release 页面；不会自动下载、挂载、替换或执行更新。

这是有意设置的安全边界：当前 DMG 为 ad-hoc 签名，尚未经过 Developer ID 公证。自动更新需要 Developer ID 签名、公证以及独立的更新签名。

## 安装

从 Release 资产下载 `OMP-API-Manager-0.3.0-macos-arm64.dmg` 和 `.sha256` 文件。将两个文件放在同一目录后运行：

```sh
shasum -a 256 -c OMP-API-Manager-0.3.0-macos-arm64.dmg.sha256
```

校验通过后打开 DMG，将应用拖入 Applications。该构建支持 Apple Silicon，需要 macOS 14 Sonoma 或更高版本。

## 验证

该版本源码通过了严格 Swift 构建和全部 31 项自动化测试。打包流程会验证应用签名，并在创建 DMG 前实际启动完成的应用。

## 可复现打包

在装有 Xcode 的 Apple Silicon Mac 上运行 `scripts/package-dmg.sh 0.3.0`。版本参数必须与 `Sources/OMPAPIManagerApp/Resources/AppVersion.txt` 一致。
