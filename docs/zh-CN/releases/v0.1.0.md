# OMP API Manager v0.1.0

> 语言： [English](../../releases/v0.1.0.md) | **简体中文**

发布于 2026-07-11。

## 第一个开发者版本

这是 OMP API Manager 面向 macOS 14+ 和 Swift 6 的第一个公开开发者版本。它提供本地优先的工作流，用于安全管理自定义服务商、运行仅 localhost 的网关和查看私密用量。

## 包含内容

- OMP 16.x 安装发现和安全配置集成。
- 带备份与冲突检测的原子 YAML 更新。
- 由 macOS Keychain 支持的服务商凭据。
- OpenAI/Anthropic 兼容服务商配置、模型获取和连接测试。
- 使用独立本地令牌的 localhost 网关与 SSE 转发。
- 脱敏 SQLite 用量记录以及 CSV/JSON 导出。
- 原生 SwiftUI 界面和 22 项自动化测试。

## 安装

从 Release 资产下载 `OMP-API-Manager-0.1.0-macos-arm64.dmg`，打开它并将应用拖到 Applications。该构建使用 ad-hoc 签名，尚未进行 Developer ID 公证。macOS 可能提示未知开发者；请只在从本仓库官方 Release 页面下载后打开。

也可以克隆带标签的源代码，在 Xcode 中打开 `Package.swift` 并运行 `OMPAPIManager` Scheme；或者在终端运行 `swift run OMPAPIManager`。

## 安全说明

API Key 保留在 macOS Keychain。应用不会持久化提示词、响应、授权头或 API Key。详见[安全政策](../../../SECURITY.zh-CN.md)和[隐私政策](../../../PRIVACY.zh-CN.md)。

## 可复现打包

在装有 Xcode 的 Apple Silicon Mac 上运行 `scripts/package-dmg.sh 0.1.0`。该命令会在 `dist/` 中创建 DMG，并输出其 SHA-256 校验和。
